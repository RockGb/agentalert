# Agent Alert 🎵

Plays a sound when your AI agent (GitHub Copilot, Cursor, Antigravity, etc.) finishes a task. No more staring at the screen waiting for generation to end!

## Features

- **Built-in Presets**: Choose from classic sounds like *Glass, Hero, Ping, Funk, Morse, and more*.
- **Custom Sound Support**: Use any audio file on your computer (.mp3, .wav, .aiff).
- **Auto-stop Duration**: Perfect for long sounds—tell it exactly how many milliseconds to play.
- **AI Heuristics**: Automatically detects when large code generations finish.
- **Standalone**: No external server or setup required. It works directly out of the box.

## Requirements

1. **Mac OS**: Currently optimized for macOS using the built-in `afplay` command.

## Setup

1. Install the extension.
2. Open Settings (`Cmd + ,`) and search for **"Agent Alert"**.
3. Configure your preferred **Preset** or **Custom Sound Path**.
4. (Optional) Set a **Playback Duration** if you want the sound to cut off early.

## Why this exists?
Coding with AI is great, but waiting for "Thinking..." to turn into code can be a distraction. This extension gives you an auditory notification so you can switch focus and come back precisely when the task is finished.
